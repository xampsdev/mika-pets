# 🐶 MIKA PETS

Site institucional para serviços de hospedagem e cuidados com pets, desenvolvido com foco em **UI/UX clara**, **performance**, **responsividade** e **código limpo**.

O projeto apresenta informações sobre a empresa, galeria interativa de pets, depoimentos de clientes e formas de contato, com navegação intuitiva e comportamento previsível em qualquer dispositivo.

# Objetivo do Projeto

- Apresentar os serviços da MIKA PETS de forma clara e confiável  
- Facilitar a navegação e o acesso às informações essenciais  
- Aplicar boas práticas de HTML, CSS e JavaScript puro  
- Servir como projeto de portfólio com padrão profissional  

---

# Tecnologias Utilizadas

- **HTML5**
  - Estrutura semântica
  - Acessibilidade básica (`aria-label`, hierarquia de headings)

- **CSS3**
  - Layout responsivo (mobile first)
  - Flexbox e Grid
  - Media Queries por breakpoint real de mercado
  - Transições e microinterações
  - Variáveis CSS

- **JavaScript (Vanilla JS)**
  - Menu mobile interativo
  - Modais dinâmicos
  - Scroll suave para navegação
  - Botão “Back to Top”
  - Destaque visual no footer ao clicar em Contato

Sem frameworks. Sem bibliotecas. Implementação mínima e controlada.

---

# Responsividade

Breakpoints utilizados:

- Mobile (base)
- Tablet retrato (481px – 768px)
- Tablet paisagem (769px – 1024px)
- Desktop / Full HD (1025px – 1920px)
- 2K (1921px – 2560px)
- 4K / TVs (2561px+)

Cada breakpoint ajusta:
- Tipografia
- Espaçamento
- Grid de conteúdo
- Escala visual

---

# Decisões de UX/UI

- Navegação simples e direta
- CTA claro para contato
- Feedback visual em interações
- Scroll guiado para reduzir esforço cognitivo
- Interface previsível (nenhum comportamento surpresa)
