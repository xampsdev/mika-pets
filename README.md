# 🐶 MIKA PETS

Landing page institucional para uma empresa fictícia de hospedagem e cuidados para pets.

O projeto foi desenvolvido com foco em **UI/UX**, **responsividade**, **acessibilidade básica** e **interações com JavaScript puro**, criando uma experiência simples, intuitiva e consistente em diferentes tamanhos de tela.

## Sobre o projeto

A MIKA PETS é um projeto de estudo e portfólio desenvolvido para simular a presença digital de uma empresa especializada em hospedagem e cuidados com pets.

A interface foi construída para apresentar informações de forma clara e facilitar o acesso às principais áreas da página:

* Informações sobre a empresa
* Serviços e diferenciais
* Galeria interativa de pets
* Depoimentos de clientes
* Newsletter
* Informações de contato

O principal foco do projeto é aplicar conceitos de **UI/UX e design responsivo** utilizando tecnologias fundamentais do desenvolvimento Front-End.

---

## Funcionalidades

* Menu mobile com botão hambúrguer
* Navegação por âncoras entre as seções
* Scroll suave entre áreas da página
* Galeria interativa de pets
* Modais com informações individuais sobre cada pet
* Seção de depoimentos de clientes
* Newsletter com validação e mensagem de sucesso
* Destaque visual temporário ao acessar seções específicas
* Botão “Back to Top” para retornar ao início da página
* Layout responsivo para diferentes tamanhos de tela
* Interface com transições e microinterações

## Tecnologias utilizadas

### HTML5

* Estrutura semântica
* Elementos organizados por seções
* Hierarquia adequada de títulos
* Acessibilidade básica com atributos `aria`

### CSS3

* Layout responsivo
* Flexbox
* CSS Grid
* Media Queries
* Variáveis CSS
* Gradientes
* Transições
* Microinterações
* Estados visuais como `:hover` e classes dinâmicas

### JavaScript

Implementação com **Vanilla JavaScript**, sem frameworks ou bibliotecas externas.

Utilizado para:

* Controle do menu mobile
* Abertura e fechamento dos modais
* Navegação com scroll suave
* Destaques temporários de seções
* Funcionamento da newsletter
* Exibição do botão “Back to Top”

## Responsividade

O projeto foi desenvolvido para se adaptar a diferentes tamanhos de tela.

Breakpoints utilizados:

* **Mobile:** até 480px
* **Tablet em retrato:** 481px – 768px
* **Tablet em paisagem:** 769px – 1024px
* **Desktop / Full HD:** 1025px – 1920px
* **2K:** 1921px – 2560px
* **4K e telas maiores:** 2561px+

Os breakpoints ajustam elementos como:

* Tipografia
* Espaçamentos
* Largura dos componentes
* Organização dos cards
* Grid da galeria
* Layout do menu
* Escala visual dos elementos

## Foco em UI/UX

As principais decisões de interface foram baseadas em:

* Navegação simples e direta
* Hierarquia visual clara
* Feedback visual após interações
* Navegação por âncoras
* Scroll suave
* Destaque temporário das seções acessadas
* Menu adaptado para dispositivos móveis
* Componentes visualmente consistentes
* Redução de esforço cognitivo durante a navegação

A interface foi construída para que o usuário consiga identificar rapidamente **onde está**, **o que pode fazer** e **como acessar as informações principais**.
